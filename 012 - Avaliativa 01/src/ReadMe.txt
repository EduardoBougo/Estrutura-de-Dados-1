Implementação Lista Duplamente Encadeada

Desenvolver uma aplicação de Lista Duplamente Encadeada que implemente uma agenda.
Os contatos dessa agenda devem ter pelo menos quadro campos de informação: CPF, nome, idade, fone e email.
Essa lista deve implementar as operações:
    * Inserir no final;
    X Inserir Ordenado pelo nome;
    X Ordenar pelo nome;
    X Ordenar pela idade;
    X Alterar pelo nome;
    X Consultar pelo nome;
    X Excluir pelo CPF;
    * Imprimir (cabeça -> cauda);
    * Imprimir Invertido (cauda -> cabeça).
Podem se basear nos códigos apresentados durante as aulas.

A aplicação deve disponibilizar um menu que permita ao usuário escolher as operações até que decida sair da apliação.
Crie as classes e implementações que julgar necessário e entregue os arquivos .java que implementar e que permitam a aplicação ser compilada e executada. Evite a redundância de código.
